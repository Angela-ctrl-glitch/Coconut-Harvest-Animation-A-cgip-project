# Coconut-Harvest-Animation-A-cgip-project
A Computer Graphics mini-project using Python (Pygame) that simulates a coconut tree harvesting animation. The project demonstrates Bresenham Line Drawing, Midpoint Circle Algorithm, Scanline Fill, Polygon Filling, and 2D Transformation with frame-based animation.


## Description
This is a Computer Graphics mini-project implemented using Python and Pygame. It simulates a coconut tree harvesting process with animation.

## Features
- Automatic animation of a coconut climber
- Coconut plucking and falling animation
- Moving crows and blinking sun
- Realistic background with mountains and paddy field

## Algorithms Used
- Bresenham Line Drawing Algorithm
- Midpoint Circle Algorithm
- Scan Line Fill Algorithm
- Polygon Filling Algorithm
- 2D Translation Transformation
- Frame-Based Animation

## Technologies
- Python
- Pygame
